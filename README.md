# Django-SimpleAPI

### Testing with Postman:

Get all:

```
http://127.0.0.1:8000/movies/
```

By ID:

```
http://127.0.0.1:8000/movies/1
```

### Running locally

Install requirements:

```sh
pip install -r requirements.txt
```

Migrate:

```sh
py manage.py migrate
```

Runserver:

```sh
py manage.py runserver
```
